## Builds

`bipolar` codes in C++17, a latest version of gcc is recommended.

[bazel](https://github.com/bazelbuild/bazel/) is required to build and run tests/benchmarks.

```bash
bazel build ...
```
to build the project.

```bash
bazel test ...
```
to run tests.
